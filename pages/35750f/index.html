# 插件推荐

  • vuepress-plugin-fulltext-search (opens new window) 全文搜索

  • vuepress-plugin-thirdparty-search (opens new window) 可以添加第三方搜索链接的搜索框

  • vuepress-plugin-one-click-copy (opens new window) 代码块一键复制

  • vuepress-plugin-comment (opens new window) 评论区

  • vuepress-plugin-vssue (opens new window) 评论区（单页）

  • vuepress-plugin-vssue-global (opens new window) 评论区（全局）

  • vuepress-plugin-smplayer (opens new window) 播放器

  • vuepress-plugin-flowchart (opens new window) 流程图

  • vuepress-plugin-mathjax (opens new window) 数学公式

  • vuepress-plugin-tabs (opens new window) 选项卡

  • vuepress-plugin-element-ui (opens new window) Element UI

  • 花里胡哨的插件 (opens new window) 鼠标点击特效、背景彩带、音乐播放器等花里胡哨的插件
